Get latest Covid-19 data

Covid-19 data on a country level

  • /covid/v1/daily/latest/all-countries.json
  • /covid/v1/daily/latest/<country-id>.json (example: /covid/v1/daily/latest/it.json)

Covid-19 data on a state level

  • /covid/v1/daily/latest/us/all-states.json
  • /covid/v1/daily/latest/us/<state-id>.json (example: /covid/v1/daily/latest/us/md.json)

Covid-19 data on a county level

  • /covid/v1/daily/latest/us/all-counties.json
  • /covid/v1/daily/latest/us/<state-id>/all-counties.json (example: /covid/v1/daily/latest/us/md/all-counties.json)
  • /covid/v1/daily/latest/us/<state-id>/<fips>.json (example: /covid/v1/daily/latest/us/md/24031.json)

Get historical Covid-19 data

To get a historical data you need:

  • Choose any url above
  • Change latest to the specific date YYYYMMDD

Examples: /covid/v1/daily/20200510/us/va.json, /covid/v1/daily/20200510/us/md/24031.json

Additional data

Get all data sources

  • /covid/v1/source/latest.json

Get executive orders

  • /covid/v1/executive-orders/latest/us/all-states.json
  • /covid/v1/executive-orders/latest/us/<state-id>.json (example: /covid/v1/executive-orders/latest/us/dc.json)

Get latest Covid-19 data

Covid-19 data on a country level

  • /covid/v{VERSION}/daily/latest/all-countries.json
  • /covid/v{VERSION}/daily/latest/<country-id>.json (example: /covid/v{VERSION}/daily/latest/it.json)

Covid-19 data on a state level

  • /covid/v{VERSION}/daily/latest/us/all-states.json
  • /covid/v{VERSION}/daily/latest/us/<state-id>.json (example: /covid/v{VERSION}/daily/latest/us/md.json)

Covid-19 data on a county level

  • /covid/v{VERSION}/daily/latest/us/all-counties.json
  • /covid/v{VERSION}/daily/latest/us/<state-id>/all-counties.json (example: /covid/v{VERSION}/daily/latest/us/md/all-counties.json)
  • /covid/v{VERSION}/daily/latest/us/<state-id>/<fips>.json (example: /covid/v{VERSION}/daily/latest/us/md/24031.json)

Get historical Covid-19 data

A full hitorical data

Covid-19 historical data on a country level

  • /covid/v{VERSION}/history/active/<country-id>.json (example: /covid/v{VERSION}/history/active/it.json)

Covid-19 historical data on a state level

  • /covid/v{VERSION}/history/active/us/<state-id>.json (example: /covid/v{VERSION}/history/active/us/md.json)

Covid-19 historical data on a county level

  • /covid/v{VERSION}/history/active/us/<state-id>/<fips>.json (example: /covid/v{VERSION}/history/active/us/md/24031.json)

Historical data on a specific day

  • Choose any url above
  • Change latest to the specific date YYYYMMDD

Examples: /covid/v{VERSION}/20200510/us/va.json, /covid/v{VERSION}/20200510/us/md/24031.json

  • /covid/v{VERSION}/executive-orders/latest/us/<state-id>.json (example: /covid/v{VERSION}/executive-orders/latest/us/dc.json)
